Microsoft OFFICE LTS 2021 PLUS

Microsoft Office LTS 2021 Plus

Up 10 Oktober 2024

Download Microsoft Office 2021 Full Version Gratis. Sebagai produk penerus dari MS Office, tentunya update terbaru ini membawa banyak perubahan yang sangat dinantikan pengguna. Software ini dirilis oleh Microsoft pada bulan Oktober 2021 lalu dan memiliki banyak fitur terbaru seperti Inking tools, Functions, Data Types, Editing Tools, Motion Graphics dan masih banyak lainnya. Semua pengguna Windows 11 dan MacOS Monterey dapat menggunakan software ini dengan semua fitur yang gratis.

Microsoft Office LTS 2021 telah diperbarui dengan desain user interface yang sesuai dengan Windows 11. Desainnya terasa lebih sederhana dan menawarkan pengguna pengalaman yang lebih modern. Dalam tampilannya, aplikasi menampilkan skema warna netral dan sudut yang lebih lembut, menciptakan tata letak yang minimalis dan ramah pengguna. Kalian juga bisa install office 2021 64 bit atau 32 bit tergantung dengan sistem operasi yang digunakan.

Banyak fitur terbaru yang disematkan pada perilisan kali ini. Termasuk Microsoft Excel yang sekarang sudah memiliki fungsi XLOOKUP dan XMATCH, dua fitur memang sangat dinantikan oleh pengguna di seluruh dunia. Selain itu, PowerPoint juga memiliki kemampuan untuk perekaman saat melakukan slide show. Dukungan untuk file OpenDocument (ODF) juga lebih cepat dan lebih baik.


Fitur Aplikasi Microsoft Office LTS 2021 Plus

  • Perubahan User Interface yang lebih modern seperti Windows 11 Pro
  • Terintegrasi dengan Microsoft Teams (Screen Share dan lainnya lebih mudah)
  • Pembaruan Microsoft Excel 2021 dengan fitur XLOOKUP dan XMATCH
  • Kemampuan untuk Record Slide Show di Power Point 2021
  • Melakukan instant search di Microsoft Outlook
  • Peningkatan performa dari versi Office sebelumnya
  • Perbaikan bugs dan beberapa peningkatan fitur
  • Sudah mendukung fitur Dark Mode di Windows 11
  • Microsoft Word mendukung Line Focus untuk reader
  • Draw Tab diperbaharui dengan fitur dan tampilan baru
  • Mendukung OpenDocument format (ODF)
  • Berjalan dengan baik di Windows 11 Pro paling terbaru
  • Include Microsoft Visio 2021 dan Microsoft Project 2021

Instalasi

  1. Buka Command Prompt sebagai Administrator:
    • Klik tombol Start, ketik "cmd" di kolom pencarian.
    • Klik kanan pada aplikasi Command Prompt yang muncul, kemudian pilih "Run as administrator".
  2. Jalankan Perintah Konfigurasi di CMD:
    • Arahkan Command Prompt ke folder tempat file instalasi berada. Misalnya: cd C:\Users\nama-user-anda\path-file yang anda simpan\OFFICE LTS 2021 PLUS
    • Copy dan jalankan perintah berikut untuk memulai instalasi: setup.exe /configure configuration.xml
  3. Tunggu Proses Instalasi Selesai:
    • Instalasi akan dimulai secara otomatis. Tunggu hingga proses selesai.
    • Setelah selesai, Office 2021 siap digunakan.
