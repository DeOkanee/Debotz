Instalasi IBM SPSS v26 Statistic

  1. Download SPSS 26 Full Version pada link di bawah
  2. Ekstrak dengan WinRAR 7.01 terbaru
  3. Setelah terextrak buka folder "SPSS v26" lalu buka folder "Setup" Jalankan "setup.exe" installernya , pilih IBM SPSS Statistic
  4. Setelah selesai, jangan dibuka dulu
  5. Buka Folder "Jamu" lalu Copy file "lservrc" yang tersedia
  6. Paste and replace:
    • Arahkan ke folder tempat file instalasi berada. Misalnya:
      • C:\Program Files\IBM\SPSS\Statistics\26
    • atau kalau tidak ada di folder tersebut, arahkan ke folder ini
      • C:\Program Files (x86)\IBM\SPSS\Statistics\26
    • atau juga bisa paste pada file "PASTE JAMU DISINI"
  7. Pastikan file aslinya sudah tertimpa "Paste dan Replace"
  8. Lalu buka aplikasinya Ketik di pencarian SPSS, Done enjoyyy
